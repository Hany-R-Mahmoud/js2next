# Product

## Register

product

## Users

Frontend developers learning or strengthening JavaScript, React, and Next.js. They use the product in focused study sessions to understand concepts, practice applying them, and return to weak areas without losing context.

## Product Purpose

JS2Next connects JavaScript foundations to React and Next.js decisions through source-backed lessons, deliberate practice, review, progress, and assessments. Success means a learner can explain the layer underneath a pattern, apply it in code, and see what to study next.

## Brand Personality

Encouraging, Socratic, rigorous. The interface should feel like a focused learning studio: clear enough for a first session, specific enough for serious practice, and calm when a learner is stuck.

## Anti-references

Avoid generic SaaS dashboard polish, noisy gamification, decorative motion, low-contrast dark mode, dense card grids, and copy that promises mastery without showing the next useful action.

## Design Principles

- Make the next learning action obvious.
- Preserve the learner's reasoning and context across lessons, practice, and review.
- Use progressive disclosure for technical complexity.
- Treat progress as feedback, not decoration.
- Prefer specific, source-backed language over motivational filler.

## Accessibility & Inclusion

Target WCAG 2.2 AA. Preserve keyboard access, visible focus, semantic landmarks, 44px touch targets, readable contrast, text resizing, reduced motion, and no horizontal overflow at 320px. Never rely on color alone for progress, mastery, lock, or feedback states.
