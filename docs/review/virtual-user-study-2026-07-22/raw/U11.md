## Severity-tagged issues

### P1 — Primary navigation clicks do not navigate reliably

From Tracks, clicking Learn stayed on `/tracks`; from cumulative assessment, Search stayed on the assessment page; at mobile Settings, Home stayed on Settings while marking Home active. Direct URLs worked.

### P1 — Search unavailable in mobile navigation

At 390×844 primary navigation contained only Home, Review, Progress, Settings; no Search link/menu.

### P2 — Onboarding does not capture persona level or learning style

Onboarding asked only for name and showed Beginner; Settings initially showed Beginner/Active/React fundamentals.

### P2 — Arabic localization readiness is low

Arabic name/reflections worked, but Arabic search returned no localized results and visible content remained English.

No P0 confirmed.
