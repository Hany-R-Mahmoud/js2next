## Issues

Home from Tracks, Open track review, and Back to learning retained current URL/content instead of navigating; direct URLs worked.

### P2 — Intermediate learner is assigned Beginner path

Onboarding showed Welcome, ليان and Beginner path; expected level capture or explanation.

### P2 — Arabic search has no localized discovery

Arabic `دوال` returned 0/79 and no canonical matches; English `closures` returned one result.

### P2 — RTL/bidi readiness is weak

Arabic saved successfully, but DOM evidence showed `lang="en"`, no `dir`, and Arabic textareas computed LTR.

### P2 — Progress/state persistence is unclear

JS-M11 mastery and topic states changed across navigation/reload while module Check stayed locked.

No P0 confirmed.

## Positives / accessibility / evidence

Strong Learn → Practice → Check → Reflect model, clear accessible roles/headings/tabs/radios/textboxes/progress bars, immediate feedback, explicit mastery threshold, Arabic text entry/save, and visible local progress. Evidence included JS-M09 assessment 100%, saved reflection, Arabic versus English search, JS-M11 Practice 6/6 with Check disabled, and cumulative entry with 24 questions. Navigation labeling was good despite unreliable transitions.

## Confidence / limitations / continue decision

JS-M09 and shared-surface findings high; JS-M11 state medium; R-M08/Review/Progress/Settings not tested. No screenshots, no cumulative submission, no optional topic, no full RTL audit. Continue JS-M09; pause reliance on later modules until navigation, persistence, Arabic search, and RTL improve.
