## Limitations / continue decision

R-M06/N-M04 partial, React cumulative not opened, no screenshots, no reset/destructive paths, no full RTL/AT/keyboard audit. Conditional continue: JS-M11 is usable, but hold broader rollout until profile persistence and responsive Search access are fixed.
