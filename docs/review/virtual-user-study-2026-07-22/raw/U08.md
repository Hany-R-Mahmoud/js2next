## Severity-Tagged Issues

### P2 — Arabic search returns no localized or transliterated results

- URL: `/home`; entered `إغلاق`.
- Observed: `Showing 0 of 79 topics. No canonical topics match this search.`
- Expected: localized matches, transliteration support, or an explicit English-only search hint.
- Impact: Arabic-first learners may not find relevant content.
- Translation suggestion: add Arabic aliases for core concepts, such as `إغلاقات / closures`, and bilingual result labels.

### P2 — Initial onboarding profile does not match the learner persona

- URLs: `/` and `/settings`.
- Observed after onboarding: `Beginner path`; Settings initially showed `Beginner` and `Active`.
- Expected: Intermediate + Reflective learners should be asked for or assigned the corresponding profile.
- Mitigation: Settings allowed changing to Intermediate + Reflective, and persistence worked.

### P2 — Mobile bottom navigation labels appear visually crowded

- Viewport: 375×812.
- Observed: bottom navigation labels visually abutted as `HomeReview`.
- Expected: clear separation between labels.
- DOM still exposed distinct accessible links Home, Review, Progress, and Settings.

No P0 or P1 issues observed.
