## Teach-back

OOP groups state with behavior. JavaScript supports classes, factories, constructor functions, and prototype delegation. `Object.create()` delegates through the prototype chain; `extends` links prototypes; `super()` initializes parent state. Choose the smallest pattern preserving ownership and invariants.
