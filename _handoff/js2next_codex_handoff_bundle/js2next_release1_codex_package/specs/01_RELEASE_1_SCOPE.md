# Release 1 Scope

## Included

- Canonical repository: `js2next`.
- Next.js/React/TypeScript/Tailwind application foundation.
- Three ordered learning tracks.
- Track, module, topic, and subtopic navigation.
- Markdown-first content packets compiled or normalized to JSON.
- Draft preview and manifest-controlled publication.
- In-lesson MCQs, topic quizzes, module reviews, and cumulative reviews.
- Explanations, hints, per-choice feedback, retries, and review recommendations.
- 80% topic/module mastery target.
- Required, optional, advanced, and orientation placements.
- Soft prerequisite guidance.
- Local progress, local review queue, and export/import.
- Responsive behavior and existing accessibility preferences.
- Structural validation, unit tests, route tests, and E2E critical-flow tests.
- Legacy instructional archive and reversible cutover.

## Excluded

- Authentication and user accounts.
- Server database and cross-device progress.
- Code execution or automated code grading.
- Full CMS.
- Payments, subscriptions, certificates, cohorts, social features, notifications.
- External analytics platform.
- Adaptive ML/AI recommendations.
- Automatic publication.

## Completion definition

Release 1 is complete when a fresh local learner can:

1. Open the platform and select one of the three tracks.
2. Browse ordered modules and topics.
3. Read a new canonical topic packet.
4. Answer in-lesson MCQs and receive feedback.
5. Complete a topic quiz, retry it, and reach 80% mastery.
6. Complete a module review.
7. See local progress and review recommendations.
8. Export progress, clear storage, import the export, and recover progress.
9. Never encounter legacy learning content through the new runtime catalog.
