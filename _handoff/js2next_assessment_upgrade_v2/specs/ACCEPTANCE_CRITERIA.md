# Acceptance Criteria

- Exactly 79 topic assessment profiles are active.
- Exactly 632 v2 questions are active and IDs are unique.
- Every question has four choices and exactly one correct choice.
- Every question is draft and pending human review.
- Every coding topic has at least four code-presented questions in Q04–Q08.
- Every hybrid topic has at least three code-presented questions in Q04–Q08.
- Workflow topics use realistic scenarios and available artifacts rather than definition trivia.
- Conceptual topics are not forced to display irrelevant code.
- Module and cumulative assessment references resolve to v2 questions.
- Code questions render through the application code-block component.
- Choice text wraps and remains accessible on mobile.
- Explanation and choice feedback appear after submission.
- Mastery threshold remains 80% and retries remain unlimited.
- Lesson content and curriculum order are unchanged by this focused patch.
- V1 assessment attempts are reversibly archived; lesson progress is preserved.
- No v1 question is reachable from the active catalog.
- No v2 question is automatically marked reviewed or published.
- Existing and new quality commands pass.
